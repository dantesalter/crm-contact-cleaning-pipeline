# CRM Contact Cleaning Pipeline

Python/Pandas pipeline for cleaning, normalizing, and enriching CRM contact data.

## Features
- Email normalization
- Phone number formatting
- Domain extraction
- Organization mapping
- Contact anonymization
- CSV export workflow

## Tech Stack
- Python
- Pandas
- Regex
- Faker

## Data Privacy
All uploaded datasets have been anonymized to preserve privacy while maintaining structural integrity for demonstration purposes.
